HTML

HTML (Hypertext Markup Language) is basically the skeleton or the back bone of any website. HTML is used to sctructure and set up the page to apply elements to it.

CSS

CSS (cascading style sheet) Css is used to style things if you couldn't tell by the name already. It basically takes the items from the HTML document and adds attributes to them wether it be background color, font size, size of the area, shape, borders, virtually everything.

JavaScript

Js not to be confused with java is another language you can use to manipulate objects in css or html and could replace both or just play as a supportive role. javascript is a very powerful language and it can be used in virtually any platform.
